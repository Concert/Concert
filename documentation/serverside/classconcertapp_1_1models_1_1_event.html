Detailed Description

create_concert_user is bound to the post_save signal.

So everytime a model object gets saved, the create_concert_user does stuff. An abstract class (abstract by Concert semantics, not Django) used as a base event class. Not abstract so models can have lists of events (i.e. a `User` can have a m2m relationship of unread events)
